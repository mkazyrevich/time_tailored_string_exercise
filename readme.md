## How to use this repository

1. **Install Node.js**

2. **Install all dependencies**
```
npm install
```

3. **Calculate spent time for each user fron file**
```
node <path to calculateSpentTimeFromFile.js> <path to source file> <path to target file>
```

4. **Run all tests**
```
npm test
```
